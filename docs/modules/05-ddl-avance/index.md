# Module 5 — DDL avancé

## Objectifs du module
- Comprendre les opérations avancées de définition de schéma avec `ALTER TABLE`.
- Gérer les suppressions et mises à jour en cascade et leurs conséquences sur l'intégrité référentielle.
- Créer et optimiser des index de colonne et comprendre leur impact sur les performances.
- Gérer les comptes et privilèges utilisateurs dans le SGBD.
- Appliquer des techniques sécurisées de gestion des mots de passe (hachage, salage).
- Réviser les notions clés et préparer l'examen 2.
