# Module 2 — DDL de base

## Objectifs du module
- Expliquer le rôle du DDL (Data Definition Language) dans la conception d’une base de données.
- Créer une base de données et définir sa structure à l’aide d’instructions SQL.
- Définir des tables avec des types de données appropriés.
- Appliquer des contraintes simples pour assurer l’intégrité des données.
- Mettre en place des clés primaires et des clés étrangères afin de représenter les relations.
- Modifier ou supprimer une structure de base de données de façon contrôlée.
- Importer et exporter des données simples dans une base existante.
